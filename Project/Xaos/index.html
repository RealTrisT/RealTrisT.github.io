Xaos

Xaos is a modular overlay. It has base modules for basic functionality such as ui.

It's name was insipired by greek mythology.

The first thing in greek mythology was chaos (or khaos), which originated from the separation of heaven and earth. Xaos is just how it's pronounced in some languages.

The point of this overlay is to enable anyone to write a module or a script, making use of exported functions from other modules, and progressively making such an easier process.

As an example, at some point I aim to be able to code a simple antivirus using things such as a kernel module as well as an ui module, and maybe even do it in just javascript! (for fun that is, of course the final version would be compiled).
